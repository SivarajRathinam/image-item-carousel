# Image item carousel using react js

## Installation
  ```
  git clone https://github.com/SivarajRathinam/image_item_carousel.git # or clone your own fork
  cd image_item_carousel
  npm run install or yarn
  npm run start or yarn start
  ```
##  For Live Production
  ```
  npm run build
  ```
